C — Dashboard "Multi-Projet Radar"

Remplace /dashboard par une vue multi-projet dense. Chaque projet = 1 card avec indicateur santé (vert/orange/rouge), compteurs (urgents, retards, budget). Clic = drill-down. Tu vois tes 8 chantiers d'un coup d'oeil. Les actions restent dans les modules respectifs.

Mon avis d'expert construction : L'option A est la plus ambitieuse mais la plus impactante — elle fusionne briefing + mail + tâches en un seul écran de décision. C'est ce que font les meilleurs outils de pilotage (Monday.com, Notion inbox). L'option B est le quick-win le plus réaliste (améliorer l'existant). L'option C résout un autre problème (vue macro multi-projets) mais ne réduit pas les clics sur les actions quotidiennes.
